# CovidDKDash
A dashboard displaying the current status of Covid-19 infections in Denmark. 
The data used is updated every weekday at 15:00 (3:00 PM) with the public data from Statens Serum Institut (SSI, English: the State Serum Institute).
The data can be found [here](https://covid19.ssi.dk/overvagningsdata/download-fil-med-overvaagningdata).

The dashboard show key numbers like number of tests, infected, deaths and admitted people. 
Several graphs are created showing the daily infected, admitted people, etc., for the user to explore freely.

Furthermore, the total number of infected of each municipality is showed using a map. The top 20 most infected municipalities are plotted besides it.
